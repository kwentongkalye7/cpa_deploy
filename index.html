01
BIR Tax Compliance Automation System
Built a comprehensive automation system for Philippine Bureau of Internal Revenue (BIR) compliance workflows. The system automatically processes DAT file submissions, searches for BIR confirmation emails, extracts HTML content, converts to PDF with proper naming conventions, and uploads to Google Drive folders. Overcame challenges including Gmail thread splitting, PDF rendering fidelity, and cached file duplication issues. Implemented cross-reference validation by TIN and file type (QAP, SLSP, SAWT) to ensure data integrity. This automation reduced manual processing from hours to minutes, handling hundreds of client submissions seamlessly.

02
Self-Hosted AI HR Agent
Developed a privacy-focused AI HR agent for resume-job description matching, running entirely on-premises. Built with Dockerized Ollama REST API using quantized Llama 3 (8B), integrated via LangChain in Flask. Implemented PyPDFLoader for resume parsing, SentenceTransformers for embeddings, and FAISS-CPU as vector store. The system generates AI-driven candidate fit evaluations while maintaining strict data confidentiality - crucial for our firm's requirements. Currently enhancing with quantitative match scoring and exploring more capable language models while prioritizing the balance between performance and privacy.
Llama 3 8B Docker LangChain FAISS Flask

04
Philippine Tax Assistant - LLM Fine-Tuning & RAG
Pioneered local LLM fine-tuning for Philippine tax assistance on M3 Mac (16GB RAM). Started with Gemma 2B and TinyLlama 1.1B, navigating technical challenges including gradient checkpointing, NaN losses, and catastrophic forgetting with limited hardware. Evolved to implement RAG architecture using quantized Llama 3 8B, LangChain pipeline, and FAISS vector database. This progression from pure fine-tuning to RAG demonstrated critical lessons about factual accuracy limitations in constrained environments and the importance of retrieval-augmented approaches for domain-specific applications.
Gemma 2B Llama 3 RAG LangChain FAISS
